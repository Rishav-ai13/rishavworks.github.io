FRONTEND DEVELOPMENT

Frontend Development is the development of the graphical user interface of a wedsite, through the use of HTML, CSS, and JavaScript so that users can view and interact with the website.

  • HTML:It stands for HyperText Markup Language and it is used to create webpages. It uses HTML tags and attributes to describe the structure and formatting of a web page. HTML consists of various elements, that are responsible for telling search engines how to display page content. For example, headings, lists, images, links, and more.

  • CSS:It is the language we use to style a Web page. CSS saves a lot of work. It can control the layout of multiple web pages all at once CSS Demo - One HTML Page - Multiple Styles! Here we will show one HTML page displayed with four different stylesheets.
